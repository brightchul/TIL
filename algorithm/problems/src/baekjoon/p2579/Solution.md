# 백준 온라인 저지 : 계단 오르기(2579)

## 문제 설명 [[링크]](https://www.acmicpc.net/problem/2579)

계단 오르기 게임은 계단 아래 시작점부터 계단 꼭대기에 위치한 도착점까지 가는 게임이다. 각각의 계단에는 일정한 점수가 쓰여 있는데 계단을 밟으면 그 계단에 쓰여 있는 점수를 얻게 된다.

계단 오르는 데는 다음과 같은 규칙이 있다.

1. 계단은 한 번에 한 계단씩 또는 두 계단씩 오를 수 있다. 즉, 한 계단을 밟으면서 이어서 다음 계단이나, 다음 다음 계단으로 오를 수 있다.
2. 연속된 세 개의 계단을 모두 밟아서는 안 된다. 단, 시작점은 계단에 포함되지 않는다.
3. 마지막 도착 계단은 반드시 밟아야 한다.

따라서 첫 번째 계단을 밟고 이어 두 번째 계단이나, 세 번째 계단으로 오를 수 있다. 하지만, 첫 번째 계단을 밟고 이어 네 번째 계단으로 올라가거나, 첫 번째, 두 번째, 세 번째 계단을 연속해서 모두 밟을 수는 없다.

각 계단에 쓰여 있는 점수가 주어질 때 이 게임에서 얻을 수 있는 총 점수의 최댓값을 구하는 프로그램을 작성하시오.



```
[입력]
입력의 첫째 줄에 계단의 개수가 주어진다.
둘째 줄부터 한 줄에 하나씩 제일 아래에 놓인 계단부터 순서대로 각 계단에 쓰여 있는 점수가 주어진다. 
계단의 개수는 300이하의 자연수이고, 계단에 쓰여 있는 점수는 10,000이하의 자연수이다.

[출력]
첫째 줄에 계단 오르기 게임에서 얻을 수 있는 총 점수의 최댓값을 출력한다.

[예시]
6
10
20
15
25
10
20

return 75
```

​    

## 문제풀이

DP 문제이다. 문제를 잘 보면 3개의 계단을 연달아 밟을수는 없다고 되어 있다. 
패턴을 찾기 위해서 발을 딛은 곳을 1, 아닌 곳을 0으로 처리해서 계단 각각의 칸에 올라갔을 때, 그 칸에 올라갈 수 있는 경우의 수를 조금 적어보자.

```
      1     1칸

    1 0     2칸
    1 1

  1 0 1     3칸
  1 1 0

1 0 1 0     4칸
1 0 1 1
1 1 0 1
```

패턴이 보이는가? 위의 값들로 봤을 때 `1 1 1`이 오지 않으면 된다. 2칸 올라올 때에는 기본적으로 `1 0` 이 앞에 붙는다. 1칸 올라올 때에는 `1`이 앞에 바로 붙기 때문에 이전 칸에서 `1 1` 을 피해야 한다.

모든 값들을 살펴보면 알겠지만, 모든 값들의 앞에는 `1`이 붙는다. 왜냐하면 자신의 계단에 발을 딛기 때문이다. 그렇다면 다음의 경우들을 생각해 볼 수 있을 것이다.

- 이전 칸에서 1칸 올라온 모든 경우는 전부다 앞이 `1 1 `이 된다.  따라서 다음 2칸 올라갈 때만 보낸다.
- 이전 칸에서 2칸 올라온 모든 경우는 전부다 앞이 `1 0`이 된다.  따라서 1칸, 2칸 올라갈 때 전부 적용 가능하다.

또한 각각의 칸에서는 1칸 올라왔거나, 2칸 올라왔을 때인 2가지 경우의 수만 받는다. 최대값만 받으면 되기 때문이다. 이 두가지를 기억하고 코드를 구현해보자.

​    

## 코드 구현

- 1칸, 2칸 올라오는 경우에 따라서 개별적으로 최댓값을 저장해준다. 

```java

public class Main {
    private static final BufferedReader br = new BufferedReader(new InputStreamReader(System.in));

    public static void main(String[] args) throws IOException {
        System.out.println(solution());
    }

    public static int solution() throws IOException {
        int idx = 1, len = inputValue();

        // 편의를 위해서 +1를 더해서 인덱스를 1 ~ len으로 맞췄다.
        int[] arr = new int[len + 1];
        int[][] cache = new int[len + 1][2];

        while (idx <= len) arr[idx++] = inputValue();

        idx = 1;
        cache[1][0] = cache[1][1] = arr[1];

        while (++idx <= len) {
            // -1칸에서 오르는 상황 : -1칸의 [1]을 사용한다.
            cache[idx][0] = arr[idx] + cache[idx - 1][1];
            
            // -2칸에서 오르는 상황 : -2칸의 두 값중 최댓값을 가져온다.
            cache[idx][1] = arr[idx] + getMax(cache[idx - 2]);
        }
        return getMax(cache[idx - 1]);
    }

    public static int getMax(int[] arr) {
        return Math.max(arr[0], arr[1]);
    }

    public static int inputValue() throws IOException {
        return Integer.parseInt(br.readLine());
    }
}
```
